## How to run

1. Clone this project from git.
2. Run `yarn install`.
3. Run `json-server --watch db.json --port 3003`
4. Run `yarn start`.

Application will start at http://localhost:3000.

## Technnologies & libraries: 

- React.js
- Bootstrap

## API JSON:
